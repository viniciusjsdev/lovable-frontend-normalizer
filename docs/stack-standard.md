# Stack Standard

This factory optimizes for fast MVP and prototype delivery.

## Frontend

- React
- TypeScript
- npm as the default package manager
- Vite when generated by Lovable or already present
- TanStack where already present
- Tailwind where already present
- service boundaries for mocks and future API calls

## Backend

- Django
- Django REST Framework when API endpoints are needed
- PostgreSQL-compatible database
- environment-driven settings
- Models, Services, Selectors, API Views and DTOs/Serializers

## Infrastructure

- Docker Compose for local full-stack development
- backend prepared as a container
- frontend prepared for native Vercel deployment from `frontend/`
- Supabase as managed Postgres/Auth/Storage when configured
- VPS/full-stack container deployment when needed

## Non-Goals

- Do not force one deployment platform for every MVP.
- Do not migrate frontend frameworks unless explicitly requested.
- Do not replace Lovable UI work with a simpler redesign.
